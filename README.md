
# SNN Tickets

A WordPress plugin for simple ticket management.

## Features
- Create and manage support tickets from the WordPress dashboard
- Admin and user ticket views
- Ticket status updates (open, closed, pending)
- Email notifications for ticket activity
- Customizable ticket form
- User access control

## How It Works
1. **Install & Activate**: Upload `snn-tickets.php` to your plugins directory and activate it in the WordPress admin.
2. **Ticket Submission**: Users can submit tickets via a form on the site or dashboard.
3. **Admin Management**: Admins view, reply, and update ticket statuses from the dashboard.
4. **Notifications**: Email alerts are sent for new tickets and replies.
5. **Status Tracking**: Tickets can be marked as open, closed, or pending for easy tracking.

## Usage
- Access the ticket system from the WordPress admin menu.
- Configure settings as needed for your workflow.
- Users and admins interact with tickets through the provided interface.
